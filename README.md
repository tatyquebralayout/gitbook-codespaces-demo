# GitBook Codespaces Demo

This repository demonstrates how to create an **interactive** tutorial using GitBook in GitHub Codespaces. The book walks you through the setup process and includes a small Python lab you can run directly inside the Codespace terminal.

## Quick Start

1. Open this repo in a Codespace.
2. Launch the GitBook preview with `gitbook serve`.
3. Follow the tutorial pages and run the commands from your terminal.
4. Experiment with the code in `lab/analyze.py` to see the results immediately.

The goal is to show how easy it is to combine documentation and hands-on exercises in a single environment.
