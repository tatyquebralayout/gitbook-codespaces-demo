# Getting Started

This page shows how to preview the book and interact with the example code.

1. **Open the repository in Codespaces.**
2. In the terminal, start the preview server:
   ```bash
   gitbook serve
   ```
3. When the port is forwarded, open the preview in your browser.
4. Follow the steps in each chapter and run the provided commands directly in the Codespace terminal.
5. Try running the lab script:
   ```bash
   python lab/analyze.py
   ```
   You should see the calculated average and the highest score printed in the terminal.

Working through the tutorial this way lets you read the instructions and immediately test them, creating a smooth interactive experience.
